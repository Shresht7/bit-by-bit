Bases

While binary (base 2) is the foundation of digital computing, other bases like octal (base-8), hexadecimal (base-16) and base-64 are also commonly used to represent large amounts of data.

Binary (Base 2)

Binary is a base-2 number system that uses only two symbols, typically 0 and 1, to represent numbers. Each binary digit (bit) represents a power of 2, with the rightmost bit representing 2^0 (1), the next bit representing 2^1 (2), then 2^2 (4), and so on, doubling with each position to the left.

Octal (Base 8)

Octal is a base-8 number system that uses digits from 0 to 7. Each octal digit represents three binary bits, making it a more compact way to represent binary numbers.

Decimal (Base 10)

Decimal is a base-10 number system that uses digits from 0 to 9. Each decimal digit represents a power of 10, with the rightmost digit representing 10^0 (1), the next digit representing 10^1 (10), then 10^2 (100), and so on, multiplying by 10 with each position to the left.

Hexadecimal (Base 16)

Hexadecimal is a base-16 number system that uses digits from 0 to 9 and letters from A to F (where A represents 10, B represents 11, C represents 12, D represents 13, E represents 14, and F represents 15). Each hexadecimal digit represents four binary bits, making it an even more compact way to represent binary numbers.

Base 32

Base 32 is a number system that uses 32 distinct symbols, typically the digits 0-9 and the letters A-V. Each base-32 digit represents five binary bits, providing a more compact representation of binary numbers compared to octal and hexadecimal.

Base 64

Base 64 is a number system that uses 64 distinct symbols, typically the digits 0-9, the letters A-Z, and the letters a-z, along with two additional symbols (often + and /). Each base-64 digit represents six binary bits, making it an even more compact way to represent binary numbers compared to octal, hexadecimal, and base-32.

Grouping Bits

A number-system can have any base, but having the base be a power of 2 (like 8 or 16) makes it align neatly with binary representation by simply grouping several bits together. As they are powers of 2 themselves, they provide a more compact representation of binary numbers, making them easier to read and work with. Groups of three binary bits can be represented by a single octal digit, and groups of four binary bits can be represented by a single hexadecimal digit.
