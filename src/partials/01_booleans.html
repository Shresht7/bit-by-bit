Booleans

In computing, we often need to represent and manipulate logical values, which can be either true or false. A Boolean value can be represented using a single bit, where 0 typically represents false and 1 represents true.

Boolean Values

A Boolean is a value that can only be in one of two states. Depending on the context, we might describe those two states as true and false, yes and no, or on and off. The important part is not the label, but the fact that there are exactly two possible outcomes.

This makes Booleans a natural way to represent conditions and decisions. Is a button pressed? Is a user signed in? Did two values match? Questions like these do not need a large range of numbers. They only need a way to say "yes" or "no".

In hardware and software, we often encode those two states using bits. By convention, 0 usually means false and 1 usually means true. That convention is what lets the same underlying binary representation stand in for switches, flags, comparisons, and logical expressions.

Boolean Expressions

A Boolean expression is any expression that evaluates to either true or false. A single variable like A can be Boolean, but so can a compound expression like A AND B or NOT isEmpty.

Boolean expressions are the language of decisions in computing. They control branches in code, determine which gates activate in a circuit, and decide whether a particular condition has been satisfied.

Boolean Algebra

Boolean algebra is a system for combining and transforming Boolean values. Instead of working with quantities like ordinary arithmetic, it works with truth values and operators such as NOT, AND, and OR.

These operators are used everywhere in computing. The same ideas appear in programming languages, search filters, conditional statements, and digital circuits. Later, in the logic gates section, these operators will reappear as physical gate behaviors.

Core Operators

NOT flips a Boolean value. If A is true, then NOT A is false. If A is false, then NOT A is true.

AND is true only when every input is true. An expression like A AND B succeeds only when both conditions hold at the same time.

OR is true when at least one input is true. An expression like A OR B only becomes false when both inputs are false.

XOR, or exclusive OR, is true when the inputs differ. For two inputs, it answers the question "is exactly one of these true?"
