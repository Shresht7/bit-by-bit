Introduction

A computer, is essentially an enormous collection of tiny electrical switches (called transistors). A switch can be in one of two discreet states: either on or off. This is the foundation for all digital systems.

Many things in life can be modeled using a two-state approach. Is it night or day? Is this thing on or off? yes or no? true or false? Answers to a lot of questions like these can be represented by using only two possible states.

Depending on the domain, these state can be represented differently. In computing, the most fundamental unit of information is the bit, which can have a value of either 0 or 1.
